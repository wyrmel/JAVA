ProductInfoManager

Overview:
This program allows users to input product information, including name, price, and description. The information is saved to the ProductInfo object and printed back to the user.

Functionality:
* The user specifies how many products they want to input.
* For each product, the user provides the following details:
    - Name
    - Price
    - Description
* The program saves the given information and displays it back to the user.

Output Example:
Name: Banana
Price: 2.2
Description: Yellow fruit

Name: Apple
Price: 1.5
Description: Red fruit
